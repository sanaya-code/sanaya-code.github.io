RetroArch Complete Guide for Linux

A Powerful, Cross-Platform Emulation Frontend

Overview

RetroArch is a universal emulator frontend that supports multiple gaming systems via emulation cores. It provides a unified interface for playing classic games with advanced features like:

  • Shader support (CRT filters, scanlines)
  • Netplay (online multiplayer)
  • Rewind & Fast-Forward
  • Save states & Cheat codes

Supported Systems (Popular Cores)

System Recommended Core(s) Notes
NES FCEUmm, Nestopia UE Nestopia UE has better accuracy
SNES bsnes, Snes9x Snes9x is lightweight, bsnes is cycle-accurate
Sega Genesis Genesis Plus GX Supports Sega CD & 32X
PS1 PCSX-ReARMed, Beetle PSX PCSX-ReARMed is faster, Beetle PSX is more accurate
N64 Mupen64Plus-Next Requires Vulkan for best performance
GBA mGBA, VBA-M mGBA has better compatibility
PSP PPSSPP Standalone PPSSPP may perform better

Installation on Linux

Option 1: AppImage (Portable, No Installation Needed)

  1. Download:
    • RetroArch AppImage
    • RetroArch Qt (Alternative GUI)
  2. Extract & Run: 
    chmod +x RetroArch.AppImage
./RetroArch.AppImage

Other Installation Options:

Flatpak (Recommended for Stability)

flatpak install flathub org.libretro.RetroArch
flatpak run org.libretro.RetroArch

APT (Debian/Ubuntu)

sudo apt install retroarch

Snap (Alternative)

sudo snap install retroarch

Compile from Source (Advanced Users)

git clone https://github.com/libretro/RetroArch.git
cd RetroArch
./configure && make
sudo make install

Initial Setup

Step 1: create links to saved directories

Name Command
core directory ln -s /media/newuser/data/play_game/retro_arch/configs/cores/ cores
config directory ln -s /media/newuser/data/play_game/retro_arch/configs/config/ config
auto config directory ln -s /media/newuser/data/play_game/retro_arch/configs/autoconfig autoconfig

Step 2: Update Core Info Files

  1. Open RetroArch
  2. Go to Main Menu > Online Updater > Update Core Info Files > Core Updater

Step 3: Download Cores

  1. Go to Main Menu > Online Updater > Update Core Info Files > Core Downloader
  2. Install recommended cores (see Overview table)
  3. Popular nes cores like "FCEUmm" or "Nestopia UE"
  4. Popular sega cores like Genesis Plus GX

Step 4: configure directories

Name Path
Roms Settings > Directory > File Browser
Saves Settings > Directory > Save Files
Screenshots Settings > Directory > Screenshots

GUI Customization

Menu Themes

  • Settings > Drivers > Menu
  • Ozone (Modern, recommended)
  • XMB (PlayStation-style)
  • RGUI (Lightweight, fast)

Visual Tweaks

  • Window Mode: Settings > Video > Window Mode (width:1440, height:1080)
  • Integer Scaling: ON (Pixel-perfect scaling)
  • Shader Effects: Settings > Shaders (Try CRT-Royale for retro feel)
  • Font Scaling: Settings > User Interface > Appearance (scale factor:1.30x)
  • Color Themes: Settings > User Interface > Appearance (color theme:gruvbox dark)

Save Your Configuration

After customizing, save your settings:

Main Menu > Configuration File > Save Current Configuration

Input & Controller Setup

Keyboard/Gamepad Controls

Default Keyboard: Arrow keys (Movement), Z/X (A/B), Enter (Start)

Customize Controls:

  1. Go to Settings > Input > Retropad Bindings > Port-1/2 Controls
  2. Map buttons manually or use Auto-Configuration

Hotkey Configuration

Go to Settings > Input > Hotkey to configure:

  • Hotkey Enable: Select button
  • Menu Toggle: A/4 button
  • Close Content: Start button

Recommended Hotkeys

Function Default Key Recommended Alternative
Menu Toggle F1 Select + Start
Save State F2 L2 + R2
Load State F4 L1 + R1
Fast Forward Space Right Stick Click
Reset Game F8 Select + L + R

Starting Games and Loading ROMs

Method 1: Manual Load

  1. Load Core: Main Menu > Load Core
  2. Load Content: Main Menu > Load Content > select game file

Method 2: Playlists (Auto-Scan ROMs)

  1. Go to Import Content > Scan Directory
  2. Select your ROM folder
  3. RetroArch will auto-detect games and organize them into playlists

Performance & Compatibility Tweaks

Best Launch Options

  • gamemoderun retroarch
    - Uses Feral GameMode for better performance
  • env -u WAYLAND_DISPLAY retroarch
    - Forces X11 (better compatibility)

Video Settings

  • GPU Driver: Settings > Drivers > Video â†’ Prefer Vulkan (if available)
  • Frame Delay: Settings > Latency > Frame Delay (Reduces input lag)
  • Threaded Video: ON (Helps with stuttering)

Audio Settings

  • Audio Driver: ALSA (Linux default) or PulseAudio
  • Audio Sync: OFF (Reduces latency)

Steam Deck Integration

  1. Add RetroArch as a Non-Steam Game

Advanced Features

Netplay (Online Multiplayer)

  1. Go to Main Menu > Netplay
  2. Host or Join a session
  3. Ensure both players use the same core & ROM version

Troubleshooting

Common Issues & Fixes

Problem Solution
Crashes on Launch Try --disable-wayland or switch to X11
Controller Not Detected Check dmesg for USB errors
Audio Crackling Lower Settings > Audio > Output Rate to 44100Hz
Slow Performance Enable Settings > Video > Hard GPU Sync
